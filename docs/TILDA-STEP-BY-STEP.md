# HandySeller на Tilda — пошаговая инструкция

Пошаговая сборка лендинга: копируйте текст из этого файла и вставляйте в Tilda.  
Стиль — как на app.handyseller.ru. SEO, семантика, Яндекс, Google.

---

# Часть 1. Подготовка

## Шаг 1.1. Вход в Tilda

1. Откройте https://tilda.cc
2. Нажмите «Войти»
3. Логин и пароль из `.env.secrets` (TILDA_EMAIL, TILDA_PASSWORD)

---

## Шаг 1.2. Создание сайта

1. Нажмите «Создать сайт»
2. Выберите «Пустая страница» (или «Landing» — если есть)
3. Дождитесь загрузки редактора

---

## Шаг 1.3. Настройки страницы (SEO)

1. Вверху нажмите **Настройки** (иконка шестерёнки)
2. Откройте вкладку «Параметры страницы» / «Page settings»
3. Заполните:

**Title (Заголовок страницы)** — скопируйте:
```
HandySeller — помощь мастерам в продажах на Wildberries и Ozon
```
*(до 60 символов — важно для Яндекс и Google)*

**Description (Описание)** — скопируйте:
```
Управляйте товарами на Wildberries, Ozon с телефона. Сделано для мастеров: вяжете, печатаете 3D, работаете с кожей или деревом — всё учтём.
```
*(150–160 символов — сниппет в поиске)*

**Keywords (Ключевые слова)** — скопируйте:
```
программа для мастеров, приложение для handmade бизнеса, учёт товаров для ремесленников, 3d печать на заказ, столярка на маркетплейсах, Wildberries, Ozon
```

4. Сохраните настройки.

---

## Шаг 1.4. Schema.org (микроразметка для SEO и ИИ)

Микроразметка помогает Яндекс, Google и ИИ правильно понять ваш сайт.

1. Перетащите блок **«HTML-код»** (T123) в самое начало страницы (выше всего)
2. Дважды кликните по блоку
3. Удалите всё содержимое и вставьте целиком:

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "SoftwareApplication",
  "name": "HandySeller",
  "description": "Простое приложение для мастеров, которые продают handmade на Wildberries и Ozon. Управление товарами, остатками и заказами с телефона. Подходит для вязания, 3D-печати, кожи, дерева и любых других ремёсел.",
  "applicationCategory": "BusinessApplication",
  "operatingSystem": "Web, Android, iOS (скоро)",
  "url": "https://app.handyseller.ru",
  "offers": { "@type": "Offer", "price": "0", "priceCurrency": "RUB" },
  "featureList": ["Единые остатки на всех маркетплейсах", "Одно описание для всех площадок", "Уведомления о заказах в Telegram", "Работает на телефоне"],
  "audience": { "@type": "Audience", "audienceType": "Мастера handmade, ремесленники, самозанятые" }
}
</script>
```

4. Сохраните блок.

---

## Шаг 1.5. Верификация в Яндекс.Вебмастер (handyseller.ru)

Чтобы подтвердить права на домен handyseller.ru в [Яндекс.Вебмастере](https://webmaster.yandex.ru):

1. Откройте редактор Tilda → **Настройки сайта** (иконка шестерёнки слева) → **Еще** → **Подтверждение прав**
2. Найдите поле **«Подтверждение домена для Яндекса через HTML-файл»**
3. Вставьте этот код:

```html
<html>
    <head>
        <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
    </head>
    <body>Verification: 7c228e9c02965679</body>
</html>
```

4. Сохраните настройки
5. **Опубликуйте все страницы** сайта
6. В Яндекс.Вебмастере нажмите «Подтвердить»

Если код в Вебмастере другой — замените `7c228e9c02965679` на ваш.

# Часть 13. SEO и индексация

## Яндекс

1. Добавьте сайт в Яндекс.Вебмастер: https://webmaster.yandex.ru  
2. Укажите главное зеркало: `https://handyseller.ru`  
3. Загрузите sitemap (Tilda создаёт автоматически: handyseller.ru/sitemap.xml)

## Google

1. Добавьте сайт в Google Search Console: https://search.google.com/search-console  
2. Подтвердите домен  
3. Отправьте sitemap

## Open Graph (для соцсетей)

В настройках страницы Tilda заполните:  
- OG Title = Title страницы  
- OG Description = Description  
- OG Image = скриншот или логотип (1200×630 px)
