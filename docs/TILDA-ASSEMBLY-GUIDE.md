## Порядок действий

1. Войти на [tilda.cc](https://tilda.cc) (TILDA_EMAIL, TILDA_PASSWORD из .env.secrets)
2. Создать сайт → «Пустая страница» или подходящий шаблон
3. Добавить блок «HTML-код» в начало → вставить Schema.org
4. Добавить блоки по порядку, копируя текст из этого гайда
5. Все кнопки «Войти», «Начать», «Регистрация» → `https://app.handyseller.ru/...`
6. **Настройки → Еще → Подтверждение прав** → вставить код верификации Яндекса
7. Настройки → Публикация → подключить домен handyseller.ru
8. Опубликовать
