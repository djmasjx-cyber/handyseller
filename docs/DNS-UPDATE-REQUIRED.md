# ⚠️ Обновите DNS в reg.ru для SSL

Сейчас домен **handyseller.ru** возвращает NXDOMAIN — DNS не настроен или указывает на другой IP.

**IP вашей VM:** `158.160.209.158`

## Что сделать

1. Зайдите в [reg.ru](https://www.reg.ru) → Управление доменом handyseller.ru  
2. Раздел «DNS-серверы и управление зоной» → **Изменить**
3. Добавьте или измените A-записи:
   - **@** → `158.160.209.158`
   - **www** → `158.160.209.158`
4. Сохраните и подождите распространения (5–60 мин)
5. Выполните:
   ```bash
   VM_HOST=158.160.209.158 ./scripts/retry-certbot.sh
   ```

После этого сайт будет доступен по **https://handyseller.ru**.
