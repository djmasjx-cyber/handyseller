# DNS: Tilda + app.handyseller.ru на Reg.ru

При DNS на Tilda (ns1.tildadns.com) **нельзя** добавить A-запись для `app` — Tilda поддерживает только TXT и MX.

## Решение: DNS на Reg.ru

Сделайте Reg.ru управляющим DNS и задайте все A-записи вручную.

---

## Шаг 1. IP Tilda

1. Войдите в [Tilda](https://tilda.cc) → ваш сайт  
2. **Настройки сайта** → **Домен** → handyseller.ru  
3. Посмотрите IP-адрес, который предлагает Tilda (например, `185.215.4.10` или другой)

**Важно:** используйте именно IP из этого раздела.

---

## Шаг 2. Reg.ru: вернуть свои NS

1. [reg.ru](https://www.reg.ru) → **Домены** → handyseller.ru  
2. Раздел **«DNS-серверы»** → **Изменить**  
3. Выберите **«Стандартные DNS-серверы Reg.ru»**  
   - ns1.reg.ru  
   - ns2.reg.ru  
4. Сохраните изменения  

Подождите 5–60 минут, пока обновятся NS.

---

## Шаг 3. Reg.ru: A-записи

1. После смены NS в reg.ru появится раздел **«Ресурсные записи»**  
2. **«Добавить запись»** → создайте:

| Тип | Имя (субдомен) | Значение (IP)     |
|-----|----------------|-------------------|
| A   | @              | IP из Tilda       |
| A   | www            | IP из Tilda       |
| A   | app            | 158.160.209.158   |

**Пример:**
- `@` → `185.215.4.10`
- `www` → `185.215.4.10`
- `app` → `158.160.209.158`

IP для `@` и `www` берёте строго из Tilda (Настройки → Домен).

---

## Итог

| Домен           | IP             | Сервис      |
|-----------------|----------------|-------------|
| handyseller.ru  | IP Tilda       | Лендинг     |
| www.handyseller.ru | IP Tilda    | Лендинг     |
| app.handyseller.ru | 158.160.209.158 | Приложение |

Рекомендация Tilda: использовать DNS регистратора и A-записи для подключения домена.
